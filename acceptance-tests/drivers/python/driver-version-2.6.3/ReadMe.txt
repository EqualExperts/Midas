To run PyThon Tests on your machine

1) Python should be installed on your machine .
2) Run SetupPymongo.py script to install pymongo driver using command "python SetupPymongo.py" on command line .
3) Start mongod .
4) Start midas .
5) Run the test script using command "python MidasWithPythonDriverTest.py" on command line or
   If you are using IDE , add Python as SDK and then run tests
